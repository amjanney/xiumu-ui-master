# XiuMu UI

XiuMu UI 是一个 Vite+Vue3 的组件库。

> 注意：前端圈内部练习使用，无商业用途

安装，参见[安装](installation)。

## 应用的技术

- typescript

- jsx

- vite

- vue3

## 比较完整

有超过 70 个组件，希望有兄弟们帮少写点代码。
